
<h1>Motherboard (System Board)</h1>

* Main **hardware board** of a computer.
* Connects all major components together.



<h3>Connected Components</h3>

* **CPU** → mounted directly on the motherboard.
* **RAM** → installed in memory slots.
* **Other devices** → connected through buses and slots.



<h3>Device Types</h3>

* **Built-in components**: CPU, RAM (directly attached)
* **Add-on components**: graphics card, sound card, network card (via bus)


<h3>System Variations</h3>

* **Laptop** → many components integrated.
* **Desktop / Server** → more expansion slots available.



>Key Role :  Allows **communication** between CPU, memory, and hardware devices.
